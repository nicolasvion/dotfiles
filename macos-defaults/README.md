# macOS Defaults Scripts

This repository contains a collection of scripts to customize macOS settings. Each script corresponds to a specific `defaults` command to alter system or application preferences.

## Activity Monitor

| Script | Description |
|---|---|
| `activity-monitor-open-main-window-on-launch` | Configures Activity Monitor to open its main window upon launch. |
| `activity-monitor-sort-by-cpu-usage` | Sets Activity Monitor to sort processes by CPU usage by default. |

## App Store

| Script | Description |
|---|---|
| `app-store-enable-web-inspector` | Enables the WebKit Web Inspector for the App Store. |
| `app-store-show-debug-menu` | Enables the debug menu in the App Store. |

## Bluetooth

| Script | Description |
|---|---|
| `bluetooth-increase-audio-quality` | Increases the audio quality for Bluetooth headphones/headsets. |

## Chrome

| Script | Description |
|---|---|
| `chrome-disable-print-preview` | Disables the print preview dialog in Google Chrome. |
| `chrome-disable-swipe-navigation` | Disables swipe navigation (two-finger back/forward) in Google Chrome. |
| `chrome-expand-print-dialog` | Expands the print dialog by default in Google Chrome. |

## Disk Utility

| Script | Description |
|---|---|
| `disk-utility-enable-debug-menu` | Enables the debug menu in Disk Utility for advanced options. |

## Dock

| Script | Description |
|---|---|
| `dock-autohide` | Enables the auto-hiding of the Dock. |
| `dock-disable-animations` | Disables the animations for showing and hiding the Dock. |
| `dock-enable-magnification` | Enables magnification for Dock icons. |
| `dock-hide-recents` | Hides recent applications from the Dock. |
| `dock-minimize-windows-to-application-icon` | Minimizes windows into their application's icon in the Dock. |
| `dock-only-show-active-apps` | Configures the Dock to only show active applications. |
| `dock-remove-autohide-delay` | Removes the delay for auto-hiding the Dock. |
| `dock-set-icon-size-to-36` | Sets the Dock's icon size to 36 pixels. |
| `dock-set-magnification-icon-size-to-72` | Sets the magnified icon size to 72 pixels. |

## Finder

| Script | Description |
|---|---|
| `finder-disable-ds-store-files-on-network-volumes` | Prevents the creation of `.DS_Store` files on network volumes. |
| `finder-disable-ds-store-files-on-usb-volumes` | Prevents the creation of `.DS_Store` files on USB volumes. |
| `finder-disable-warning-before-changing-an-extension` | Disables the warning that appears before changing a file's extension. |
| `finder-disable-window-animations` | Disables all animations in the Finder. |
| `finder-enable-quit-menu-item` | Adds a "Quit Finder" menu item to the Finder menu. |
| `finder-list-view-as-default-style` | Sets the default Finder view style to List View. |
| `finder-save-to-disk-by-default` | Sets save dialogs to default to saving to disk instead of iCloud. |
| `finder-show-all-files` | Shows all hidden files (those starting with a dot) in the Finder. |
| `finder-show-file-extensions` | Shows all file extensions in the Finder. |
| `finder-show-path-bar` | Makes the path bar visible at the bottom of Finder windows. |

## Global

| Script | Description |
|---|---|
| `global-enable-text-selection-in-quick-look` | Allows text selection within Quick Look previews. |
| `global-expand-print-panel-by-default` | Expands the print panel by default in all applications. |
| `global-expand-save-panel-by-default` | Expands the save panel by default in all applications. |

## Keyboard

| Script | Description |
|---|---|
| `keyboard-disable-press-and-hold-for-special-characters` | Disables the press-and-hold feature for accented characters to favor key repeat. |
| `keyboard-enable-faster-key-repeat` | Sets a faster key repeat rate. |
| `keyboard-enable-keyboard-navigation` | Enables full keyboard navigation to move focus between controls. |

## Menu Bar

## Messages

## Mission Control

## Photos

| Script | Description |

## Safari

| Script | Description |
|---|---|

| `safari-disable-preloading-top-hits` | Disables the preloading of Top Hits in the background. |
